# Instalação

```bash
git clone https://github.com/edargains/mailwarmup.git
cd mailwarmup
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
mv .env.sample .env
```

#Configuração

## Editar o arquivo .env com as informações necessárias. 

# Execução
